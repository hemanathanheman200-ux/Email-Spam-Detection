    About the machine learning stack

    Short explanations you can narrate during your presentation—pair with your actual notebook or model card for depth.

    Natural Language Processing (NLP)

    NLP turns raw email text into structured features: tokenization, normalization, n-grams, and embeddings that capture semantics beyond simple keyword lists.

    Spam classification

    Binary or multi-class models learn boundaries between ham and spam using labeled corpora. Common approaches include Naive Bayes, SVMs, and transformer-based classifiers.

    Prediction & confidence

    At inference time the model outputs a probability or score. Thresholds trigger actions (block, tag, quarantine). This demo API returns a confidence value for UI display.
